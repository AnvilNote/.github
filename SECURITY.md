# Security Policy

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you've found a security issue in any AnvilNote repository, email:

**sungpinyue@gmail.com**

Please include as much of the following as you can:

- The affected repository
- The affected version or commit
- Steps to reproduce the issue
- The impact of the issue
- A suggested fix, if you have one

## Response times

AnvilNote is in early development and maintained on a limited basis. We will do our best to respond promptly, but we cannot commit to a specific response time or SLA at this stage.

## Scope

This policy applies to all repositories under the [AnvilNote organization](https://github.com/AnvilNote).
