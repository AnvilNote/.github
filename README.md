# .github

This repository contains GitHub organization profile and shared community health files for [AnvilNote](https://github.com/AnvilNote).

It is not a product repository. It does not contain AnvilNote source code. Product code lives in the individual repositories listed in [profile/README.md](profile/README.md).

## Contents

| File or directory | Purpose |
| --- | --- |
| `profile/README.md` | Organization homepage content, shown at https://github.com/AnvilNote |
| `ISSUE_TEMPLATE/` | Shared issue templates, inherited by repositories without their own |
| `PULL_REQUEST_TEMPLATE.md` | Default pull request template |
| `CONTRIBUTING.md` | General contribution guide for the organization |
| `SECURITY.md` | Security reporting policy |
| `SUPPORT.md` | Support and help channels |

## Notes

- Files here apply organization-wide by default. Individual repositories can override any of them with their own copy in the same path.
- Changes to this repository should stay scoped to organization-level configuration and documentation, not product features.
