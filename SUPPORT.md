# Support

## Bug reports

Use the bug report template in the relevant repository. If you're not sure which repository applies, see the repository map in [CONTRIBUTING.md](CONTRIBUTING.md).

## Feature requests

Use the feature request template in the relevant repository.

## Desktop install or packaging issues

For installer, signing, notarization, or packaging problems with the desktop app, use [AnvilNote/anvilnote-desktop](https://github.com/AnvilNote/anvilnote-desktop) and the release/packaging issue template.

## General project direction

For questions about roadmap, architecture, or where the project is headed, use [AnvilNote/anvilnote](https://github.com/AnvilNote/anvilnote) once it is public.

## Security issues

Do not open a public issue for a security concern. See [SECURITY.md](SECURITY.md) for how to report it.
